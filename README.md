# Make a list 📝

App that will make a list based on image search

## Purpose 💖

Should result in a greater understanding of how to solve this using 3rd party APIs and other tools

## Install app 🐣

```
npm install
```

## Start app 🚀

- Back end

Navigate to [server](./server) folder and run

```
node index
```
